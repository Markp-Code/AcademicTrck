# UniProgress - Sistema de Seguimiento Académico

## Problema Original
Aplicación Web de Seguimiento Académico para estudiantes universitarios. Multi-universidad con gestión de carreras y pensum por administradores.

## Arquitectura
- **Frontend**: React + TailwindCSS + Recharts + React Flow
- **Backend**: FastAPI + MongoDB
- **Autenticación**: JWT (admin y estudiantes)

## Credenciales

## Funcionalidades Implementadas
1. ✅ Autenticación JWT (admin y estudiantes)
2. ✅ Multi-universidad con gestión de carreras
3. ✅ Panel de administración para CRUD de:
   - Universidades
   - Carreras por universidad
   - Materias/Pensum por carrera (formulario manual)
4. ✅ Dashboard con GPA y tiempo estimado en **AÑOS**
5. ✅ Pensum por cuatrimestres y árbol de prerrequisitos (React Flow)
6. ✅ Gestión de estados de materias con validación de prerrequisitos
7. ✅ Cálculo de GPA y honores académicos
8. ✅ Registro con selección de universidad y carrera

## Datos Precargados

- Carrera: Ingeniería en Ciberseguridad (53 materias, 185 créditos)
